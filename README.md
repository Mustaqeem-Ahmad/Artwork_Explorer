# Artwork Explorer

A React application for browsing artworks from the Art Institute of Chicago API with server-side pagination and persistent row selection.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- PrimeReact (DataTable)

## Getting Started

```sh
npm install
npm run dev
```

## Features

- Server-side paginated data table
- Persistent row selection across pages
- Custom row selection overlay
- Responsive pagination footer
